# **App Name**: Rodger Creations Portfolio

## Core Features:

- Hero Section: Visually striking hero section with clear messaging about viral growth.
- Recent Work Carousel: Showcase of video editing projects with easily updatable placeholders.
- About Section: Concise presentation of skills and experience focused on gaming content and YouTube Shorts.
- Value Proposition Block: Highlight key service benefits.
- Final Call to Action: Compelling closing section designed to drive conversions.

## Style Guidelines:

- Primary color: Bold red (#FF4136) to convey confidence and performance.
- Background color: Dark gradient from near-black to deep blue (#11111B) to create a minimalist and professional look.
- Accent color: Lighter shade of red (#FF7F77) for button highlights and subtle UI elements.
- Body and headline font: 'Poppins', sans-serif, for a modern, clean aesthetic.
- Centered layout in hero section. Horizontal carousel for recent work.
- Mobile-first responsive design, with stacked elements on smaller screens.
- Subtle carousel swipe animations (optional, implemented via CSS only).
